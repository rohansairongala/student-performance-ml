# Student Performance ML Project

This is a beginner Machine Learning project that demonstrates Python programming, CSV data handling, and basic data processing.  
The project calculates student average scores from a CSV file and saves the results to another CSV file.

---

## Features

- Reads student scores from `data/scores.csv`
- Handles invalid data gracefully (e.g., "N/A", non-numeric values)
- Calculates average scores per student
- Saves averages to `data/averages.csv`
- Built with Python 3 and Pandas
